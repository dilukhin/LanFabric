# LanFabric

LanFabric — CLI-инструмент для развёртывания изолированной VPN-сети (L3) поверх интернета на базе WireGuard / AmneziaWG.

Основная задача — быстро создать окружение, в котором пользователи взаимодействуют как в локальной сети, без доступа в интернет по умолчанию.

---

## Ключевые особенности

- Простое развёртывание одной командой
- Полностью CLI (без API и веб-интерфейсов)
- Управление через SSH
- Минимальные зависимости (только стандартная библиотека Python)
- Изоляция от интернета по умолчанию
- Централизованное управление пользователями
- Автоматическая генерация конфигураций

---

## Архитектура

```

vcli-admin.py  (клиент, Windows/Linux)
|
SSH
|
vsrv-admin.py  (сервер, Ubuntu)
|
WireGuard / AmneziaWG
|
клиенты

````

### Компоненты

- `vcli-admin.py` — клиент управления
- `vsrv-admin.py` — серверный исполнитель

Связь: SSH + SCP

---

## Требования

### Сервер

- Ubuntu 22.04 / 24.04
- root или sudo-доступ
- доступ в интернет

### Клиент

- Windows / Linux
- Python 3.12+
- установлен ssh и scp

---

## Быстрый старт

### Инициализация сервера

```bash
python vcli-admin.py --host <IP> init
````

---

### Варианты установки VPN

#### 1. По умолчанию (AmneziaWG)

```bash
python vcli-admin.py --host <IP> init
```

Если AmneziaWG недоступен:

```
ERROR: AmneziaWG недоступен. Для продолжения со стандартным WireGuard перезапустите с флагом --no-amnezia
```

---

#### 2. Стандартный WireGuard (рекомендуется)

```bash
python vcli-admin.py --host <IP> init --no-amnezia
```

## Управление пользователями

### Создание

```bash
python vcli-admin.py --host <IP> add <username>
```

---

### Список

```bash
python vcli-admin.py --host <IP> list
```

---

### Удаление

```bash
python vcli-admin.py --host <IP> delete <username> <username>
```

---

### Блокировка

```bash
python vcli-admin.py --host <IP> block <username>
```

---

### Получение конфигурации

```bash
python vcli-admin.py --host <IP> config <username>
```

## Интернет-доступ

По умолчанию отключён.

Включается флагом при создании пользователя:

```bash
--internet
```

---

## Логирование

### Формат

```
YYYY-MM-DD HH:MM:SS [CLI] LEVEL: сообщение
YYYY-MM-DD HH:MM:SS [SRV] LEVEL: сообщение
```

### Особенности

* Логи сервера транслируются в клиент в реальном времени
* Нет буферизации (используется `python -u`)
* Удобно для диагностики зависаний

---

## Диагностика

### Быстрая проверка

```bash
python vcli-admin.py --host <IP> status
```

---

### Глубокая диагностика

```bash
python vcli-admin.py --host <IP> health
```

Проверяет:

* наличие WireGuard
* состояние интерфейса wg0
* порт 51820/UDP
* iptables
* sysctl
* systemd
* базу данных

---

## Удаление окружения

```bash
python vcli-admin.py --host <IP> remove <IP>
```

## Безопасность

* Только SSH доступ
* Ключевая аутентификация VPN
* Нет паролей внутри VPN
* Изоляция от интернета

---

## Ограничения

* Нет web UI
* Нет relay (L2)
* Нет мобильных клиентов "из коробки"

---

## Известные особенности

### AmneziaWG

* Может быть недоступен в репозиториях
* Не устанавливается автоматически при ошибке
* Требует явного выбора пользователя

Рекомендуется использовать:

```bash
--no-amnezia
```
